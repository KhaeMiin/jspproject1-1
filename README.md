# jspproject1-1
jps를 활용한 스프링 부트 웹 애플리케이션 개발 예제

## 프로젝트 설정
1. Project:
	- Gradle Project
	- Language: Java 11
	- Spring Boot: 2.6.5
2.  Dependencies
	- SpringWeb
	- Thymeleaf
	- Lombok
	- Validation
	- H2 Database
	- Spring Data JPA
3. Pachaging: **Jar**
	- 내장 서버(톰캣등)을 사용하고, 'webapp'경로도 사용하지 않습니다. (내장 서버 사용에 최적화)
